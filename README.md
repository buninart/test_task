# Базовая настройка Ubuntu 22.04 под требования PCI DSS с помощью Ansible

Данный репозиторий содержит Ansible-роли для:

- применения базового профиля hardening с учётом минимальных требований PCI DSS (`base_hardening`);
- настройки хоста под приложение (пример API-сервиса) (`app_role`).

## Требования

- Ansible (рекомендуется актуальная стабильная версия, 2.16+);
- Cистема: Ubuntu 22.04 LTS;
- Доступ по SSH с правами sudo.

## Как запустить

1. Укажите IP и пользователя VM в файле `inventory` (группа `appservers`).
2. Запустите плейбук:

```bash
ansible-playbook site.yml
